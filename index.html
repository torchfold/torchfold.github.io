￼ Features Alignment Performance Benchmarks TorchFold: A PyTorch & NPU  Compatible AlphaFold3 Democratizing biomolecular modeling with a native PyTorch implementation. TorchFold overcomes the framework dependencies and restrictive licensing of AlphaFold3. We provide a complete PyTorch reimplementation optimized for both NVIDIA GPUs and Huawei Ascend NPUs, achieving state-of-the-art performance with commercially friendly open weights. View Technical Report Code (GPU) Code (NPU) Key Contributions Native PyTorch A complete migration from JAX/Haiku to native nn.Module. Enables seamless integration with the broader deep learning ecosystem. NPU Optimization Optimized for Huawei Ascend NPUs via CANN backend, featuring fused attention operators for high-performance inference. Open License We release commercially friendly model weights under the MIT license, removing barriers for industry research and deployment.
